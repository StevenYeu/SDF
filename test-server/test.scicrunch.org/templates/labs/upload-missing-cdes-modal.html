Note about Data Dictionaries and CDEs

Having a data dictionary is critical to improve the interpretability of a dataset and to comply with FAIR data principles. We are also developing the ODC common data elements (CDEs) based on the data dictionaries for datasets uploaded to the ODC-SCI.

We highly encourage you to create and upload a data dictionary for your dataset. You can upload a data dictionary on the dataset page once the upload is complete.

A data dictionary is required to request a DOI for dataset publication.
