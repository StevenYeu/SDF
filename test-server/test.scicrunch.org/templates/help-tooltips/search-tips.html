Search Tips

  1. Use quotes. Use double quotation marks (" ") around phrases to ensure they are searched exactly as is, with the words side by side in the same order, particularly if your search phrase contains short words like "of", "and", "to". Example: “Neuroscience Information Framework”.
  2. Use boolean search. Use “OR” for broadening your search by connecting two or more similar concepts (synonyms). Example: use it for including different forms of the word, “ischemia” OR “ischemic”; Use “AND” for narrowing your search by only showing the resulting records containing ALL search terms. Example: “ischemic stroke” AND “clots”; Use “NOT” in a search to exclude words from your search. Example: antibody NOT sheep
  3. Use the plus and minus signs. Add "+" or “-” in front of words to force their inclusion and/or exclusion in searches (NO space between the sign and the keyword). Example: +meat -vegetables.
  4. Use the asterisk wildcard. Use ”*” to look for variations in spelling and word form. Example: “ischem*” returns ischemia, ischemic, etc.
  5. Use autocomplete vocabulary. Using autocomplete specifies will expand your search automatically with terms from the ontology. Example: synonyms, abbreviations.
  6. Using the FIND command in the search box.

The FIND command (you can use any case) allows a user to restrict their data federation search to resources that have been annotated to some special categories. We illustrate its use through several examples.

A. find:gene Alzheimer’s (locates all resources that are annotated by the category (or concept) “gene” and within those resources searches for the term “Alzheimer’s”)

B. find:protocol * reports all (* means all) resources that have been annotated with the concept “protocol”

Currently, the special categories for the FIND command (note that the command is case insensitive) are as follows.

  • Cell (find:cell mouse)
  • Disease (find:disease lymphocyte)
  • Function (find:function kinase)
  • Organism (find:organism antigen)
  • Phenotype (find:phenotype Fox)
  • Resource (find:resource microscope)
  • Anatomy (find:anatomy imaging)
  • Molecule (find:molecule influenza)
  • Protocol (find:protocol nlx_inv_090912)
