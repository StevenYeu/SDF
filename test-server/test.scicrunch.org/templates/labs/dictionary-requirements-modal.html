Summary of Data Dictionary Requirements

  1. We recommend using the template (click to download).
  2. The data dictionary must meet these guidelines:
    1. The Data Dictionary includes the columns (must have exact spelling):
      1. VariableName
      2. Title
      3. Unit_of_Measure
      4. Description
      5. DataType
      6. PermittedValues
      7. MinimumValue
      8. MaximumValue
      9. Comments
    2. Every variable (i.e. column header) in the dataset is listed in the VariableName column of the data dictionary.
    3. Every row has VariableName, Title, and Description filled out.
    4. The Data dictionary is a ".csv" formatted file.

See the data dictionary tutorial for more information.
