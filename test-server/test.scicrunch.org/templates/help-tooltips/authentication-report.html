For NIH grant applications, dkNET can assist researchers in preparing authentication plans for cell lines or antibodies to comply with the NIH Submission Policy. dkNET created an automated tool to enable researchers to create a customized report based on the cell lines and antibodies they plan to use. View an example of authentication plan here. Learn more about how dkNET can assist in preparing the authentication report for your grant applications.


Add a resource to the authentication report (dkNET account required)

  1. Log into your existing account or create a new account.
  2. Click " Add to an authentication report"
  3. If you have created an authentication report already, click to add this resource to the existing report.
  4. To add this resource to a new authentication report, click "Create a new authentication report" and fill in the basic information. Refresh the resource report page and click to add this resource to the new report.
