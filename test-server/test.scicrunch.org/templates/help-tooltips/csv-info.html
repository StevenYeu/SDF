A CSV file (comma-separated values) is a file where each field is separated by commas.

To create a CSV from Excel:

File -> Export -> Change File Type -> CSV (Comma-delimited)

The CSV format:

The first line is the header of the dataset and each following lines are data records.

For example this table:
Subject Experiment Measurement
A EtOH 0.13
A EtOH 0.15
B EtOH 1.01
B EtOH 0.90
C ZTDO, 1mM 4.02
C ZTDO, 1mM 4.94
D ZTDO, 1mM 4.66
D ZTDO, 1mM 4.79

Would look like this in CSV format: 

Subject,Experiment,Measurement
A,EtOH,0.13
A,EtOH,0.15
B,EtOH,1.01
B,EtOH,0.90
C,"ZTDO, 1mM",4.02
C,"ZTDO, 1mM",4.94
D,"ZTDO, 1mM",4.66
D,"ZTDO, 1mM",4.79
